what is ijust?

IJust is an open-source judge system used for ACM contests.
Our system provides you with all the essential features a judge system demands. The main goal was to implement a secure and easy to use judge system for Iran University of Science and Technology (IUST).

Implementation Details

Basically, our system is based on “Client-Server” architecture, the server consists of a Model and a Controller, and the client side uses an MVC architecture as seen below.


Server:

  • The server was implemented with Flask Framework.
  • Mongo-engine ORM was used for our MongoDB database.
  • For authentication we used Redis, since it offers a better performance among others.
Web Client:
  • Our single page website was created and designed using AngularJS framework.
  • Semantic-UI, Rome.js and particle.js were also used for a more responsive and user-friendly website.
